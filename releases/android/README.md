# TaskFlow Android 客户端

## 系统要求

- Android 8.0 (API 26) 及以上

## 下载

| 文件 | 类型 | 说明 |
|------|------|------|
| `TaskFlow-debug.apk` | Debug | 开发测试用，允许明文 HTTP |
| `TaskFlow-release-unsigned.apk` | Release | 需签名后分发 |

## 安装

1. 下载 APK 到手机
2. 允许「安装未知应用」
3. 点击 APK 安装

## 卸载

Android 设置 → 应用 → TaskFlow → 卸载
